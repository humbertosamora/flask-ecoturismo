Objetivo

Este projeto foi desenvolvido como requisito parcial para a conclusão do curso de bacharelado em Sistemas de Informação na PUC Minas São Gabriel (Turma 2024/01). A API REST será utilizada como base para um sistema elaborado na disciplina Fundamentos de Sistemas de Informação. O tema abordado no projeto é o Ecoturismo, com o objetivo de promover os Objetivos para o Desenvolvimento Sustentável (ODS), incentivando práticas de turismo sustentável e conscientização ambiental.

Tecnologias

A API REST foi elaborada utilizando conceitos modernos e tecnologias robustas, incluindo:

  • Programação orientada a objetos para modularidade e reutilização de código
  • Implementação de regras de negócio para garantir a integridade dos processos
  • Banco de dados relacional MySQL para armazenamento estruturado e seguro
  • Mapeamento ORM com SQLAlchemy para abstração e manipulação de dados
  • Gerenciamento de seção e autenticação com Flask-Login
  • Utilização do framework Flask para construção da API RESTful
  • Desenvolvimento de sistemas distribuídos e integração com aplicações front-end
